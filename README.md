## 📝 写作原则

### 费曼学习法应用
1. **用比喻**: 每个抽象概念都配一个生活化的比喻
2. **从具体到抽象**: 先举例子，再总结规律
3. **简化语言**: 删除专业术语，或用人话解释
4. **自问自答**: 预设读者会困惑的地方，主动解答

### 数学推导规范
- 每一步都解释"为什么"
- 不跳过中间步骤
- 用彩色标记关键公式
- 提供数值示例验证

### 代码实践要求
- 每章都有可运行的代码
- 从NumPy手写开始，再到框架实现
- 提供逐步调试的方法
- 可视化结果
