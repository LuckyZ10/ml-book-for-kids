## 本章小结

### 🎯 核心概念

| 概念 | 解释 |
|------|------|
| Sigmoid函数 | 将任意实数映射到(0,1)的S形曲线 |
| 几率 | p/(1-p)，事件发生的相对可能性 |
| 对数几率 | log(p/(1-p))，逻辑回归的线性预测目标 |
| 最大似然估计 | 找使观察到数据概率最大的参数 |
| 决策边界 | 分类的分界线，p=0.5的位置 |
| L2正则化 | 通过惩罚大权重防止过拟合 |

### 📐 关键公式

```
Sigmoid:        σ(z) = 1 / (1 + e^(-z))
Logit:          log(p/(1-p)) = w₀ + w₁x₁ + ... + wₙxₙ
预测概率:        p = σ(w₀ + w₁x₁ + ... + wₙxₙ)
损失函数:        L = -[y·log(p) + (1-y)·log(1-p)]
梯度:           ∂L/∂wⱼ = (p - y) × xⱼ
```

### 🔑 关键代码模式

```python
# 训练循环
for each sample:
    z = dot(w, x) + b
    p = sigmoid(z)
    error = p - y
    w = w - α * error * x
    b = b - α * error
```
