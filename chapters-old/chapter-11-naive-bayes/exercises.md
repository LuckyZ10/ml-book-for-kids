### 练习11.1 贝叶斯定理基础

**题目**：
某工厂生产产品，有两条生产线A和B：
- 生产线A生产了60%的产品，次品率为2%
- 生产线B生产了40%的产品，次品率为3%

现从全部产品中随机抽取一件，发现是次品。请问这件次品来自生产线A的概率是多少？

**解答思路**：

设：
- $A$ = 产品来自生产线A
- $B$ = 产品来自生产线B
- $D$ = 产品是次品

已知：
- $P(A) = 0.6$, $P(B) = 0.4$
- $P(D|A) = 0.02$, $P(D|B) = 0.03$

求 $P(A|D)$

根据贝叶斯定理：
$$P(A|D) = \frac{P(D|A) \times P(A)}{P(D)}$$

其中：
$$P(D) = P(D|A)P(A) + P(D|B)P(B) = 0.02 \times 0.6 + 0.03 \times 0.4 = 0.012 + 0.012 = 0.024$$

因此：
$$P(A|D) = \frac{0.02 \times 0.6}{0.024} = \frac{0.012}{0.024} = 0.5$$

**答案**：这件次品来自生产线A的概率是 **50%**。

### 练习11.2 拉普拉斯平滑计算

**题目**：
在文本分类任务中，我们正在训练一个多项式朴素贝叶斯分类器。已知：
- 类别"体育"的训练文档共有1000个词
- 词汇表大小为500
- 词"足球"在"体育"类中出现了8次

请计算在以下情况下 $P(\text{足球}|\text{体育})$ 的值：

(1) 不使用平滑（最大似然估计）
(2) 使用拉普拉斯平滑（$\alpha = 1$）
(3) 使用Lidstone平滑（$\alpha = 0.5$）

**解答思路**：

(1) **不使用平滑**：
$$P(\text{足球}|\text{体育}) = \frac{N_{\text{足球},\text{体育}}}{N_{\text{体育}}} = \frac{8}{1000} = 0.008$$

(2) **拉普拉斯平滑**（$\alpha = 1$）：
$$\tilde{P}(\text{足球}|\text{体育}) = \frac{N_{\text{足球},\text{体育}} + \alpha}{N_{\text{体育}} + \alpha \times |V|} = \frac{8 + 1}{1000 + 1 \times 500} = \frac{9}{1500} = 0.006$$

(3) **Lidstone平滑**（$\alpha = 0.5$）：
$$\tilde{P}(\text{足球}|\text{体育}) = \frac{N_{\text{足球},\text{体育}} + \alpha}{N_{\text{体育}} + \alpha \times |V|} = \frac{8 + 0.5}{1000 + 0.5 \times 500} = \frac{8.5}{1250} = 0.0068$$

**答案**：
- (1) 0.008
- (2) 0.006
- (3) 0.0068

### 练习11.3 朴素贝叶斯分类决策

**题目**：
考虑一个简单的天气分类问题，预测是否适合打网球。我们有两个类别："是"（适合）和"否"（不适合）。训练得到以下概率：

**先验概率**：
- $P(\text{是}) = 0.6$
- $P(\text{否}) = 0.4$

**条件概率**（$P(\text{特征值}|\text{类别})$）：

| 特征 | 特征值 | $P(\text{值}|\text{是})$ | $P(\text{值}|\text{否})$ |
|------|--------|----------------------|----------------------|
| 天气 | 晴 | 0.3 | 0.4 |
| 温度 | 凉爽 | 0.4 | 0.2 |
| 湿度 | 正常 | 0.6 | 0.3 |
| 风力 | 弱 | 0.7 | 0.3 |

现在有一个新的天气样本：{天气=晴, 温度=凉爽, 湿度=正常, 风力=弱}。

请使用朴素贝叶斯分类器预测这个样本的类别。

**解答思路**：

根据朴素贝叶斯公式：
$$P(C_k|\mathbf{x}) \propto P(C_k) \times \prod_{i=1}^{n} P(x_i|C_k)$$

**计算类别"是"的得分**：
$$\begin{aligned}
P(\text{是}) \times P(\text{晴}|\text{是}) \times P(\text{凉爽}|\text{是}) \times P(\text{正常}|\text{是}) \times P(\text{弱}|\text{是}) \\
= 0.6 \times 0.3 \times 0.4 \times 0.6 \times 0.7 \\
= 0.6 \times 0.3 \times 0.4 \times 0.6 \times 0.7 \\
= 0.03024
\end{aligned}$$

**计算类别"否"的得分**：
$$\begin{aligned}
P(\text{否}) \times P(\text{晴}|\text{否}) \times P(\text{凉爽}|\text{否}) \times P(\text{正常}|\text{否}) \times P(\text{弱}|\text{否}) \\
= 0.4 \times 0.4 \times 0.2 \times 0.3 \times 0.3 \\
= 0.4 \times 0.4 \times 0.2 \times 0.3 \times 0.3 \\
= 0.00288
\end{aligned}$$

**比较**：
- $P(\text{是}|\mathbf{x}) \propto 0.03024$
- $P(\text{否}|\mathbf{x}) \propto 0.00288$

由于 $0.03024 > 0.00288$，样本被预测为"是"（适合打网球）。

**归一化后的概率**：
$$P(\text{是}|\mathbf{x}) = \frac{0.03024}{0.03024 + 0.00288} = \frac{0.03024}{0.03312} \approx 0.913$$

$$P(\text{否}|\mathbf{x}) = \frac{0.00288}{0.03312} \approx 0.087$$

**答案**：该样本被预测为"是"（适合打网球），概率约为91.3%。

### 练习11.4 对数似然与数值稳定性

**题目**：
在一个文本分类任务中，词汇表大小为10000。使用多项式朴素贝叶斯分类器对一个新文档进行分类。已知：
- 类别"科技"的先验概率 $P(\text{科技}) = 0.4$
- 文档中有10个词，每个词在类别"科技"中的条件概率均为0.0001

(1) 如果直接计算 $P(\text{科技}) \times \prod_{i=1}^{10} P(w_i|\text{科技})$，结果会是多少？这个计算在计算机中会遇到什么问题？

(2) 使用对数变换后，计算 $\log P(\text{科技}) + \sum_{i=1}^{10} \log P(w_i|\text{科技})$ 的值。

**解答思路**：

(1) **直接计算**：
$$\begin{aligned}
P(\text{科技}) \times \prod_{i=1}^{10} P(w_i|\text{科技}) &= 0.4 \times (0.0001)^{10} \\
&= 0.4 \times 10^{-40} \\
&= 4 \times 10^{-41}
\end{aligned}$$

**问题**：$4 \times 10^{-41}$ 远远小于计算机浮点数的最小正数（约 $10^{-308}$ 对于双精度浮点数，但 $10^{-40}$ 对于单精度浮点数来说已经是下溢）。在实际计算中，这会导致**数值下溢（underflow）**，结果被错误地计算为0。

(2) **对数变换**：
$$\begin{aligned}
\log P(\text{科技}) + \sum_{i=1}^{10} \log P(w_i|\text{科技}) &= \log(0.4) + 10 \times \log(0.0001) \\
&= -0.916 + 10 \times (-9.210) \\
&= -0.916 - 92.10 \\
&= -93.016
\end{aligned}$$

（注：使用自然对数，$\ln(0.4) \approx -0.916$，$\ln(0.0001) = \ln(10^{-4}) = -4 \times \ln(10) \approx -9.210$）

**对数计算的优势**：
- 结果 $-93.016$ 在计算机浮点数范围内可以轻松表示
- 不同类别的对数得分可以直接比较大小
- 避免了数值下溢问题

**答案**：
- (1) 直接计算结果为 $4 \times 10^{-41}$，会导致数值下溢
- (2) 对数似然约为 $-93.016$（使用自然对数）

### 练习11.5 特征相关性对朴素贝叶斯的影响

**题目**：
考虑一个二分类问题，有两个二元特征 $X_1$ 和 $X_2$。真实的数据分布如下：

| 类别 | $P(X_1=1)$ | $P(X_2=1)$ | $P(X_1=1, X_2=1)$ |
|------|------------|------------|-------------------|
| A | 0.5 | 0.5 | 0.25 |
| B | 0.5 | 0.5 | 0.5 |

对于类别A，$X_1$ 和 $X_2$ 独立；对于类别B，$X_1$ 和 $X_2$ 完全相关（当$X_1=1$时$X_2$必然为1）。

现在有一个测试样本：$X_1=1, X_2=1$。

(1) 使用朴素贝叶斯分类器（假设特征独立），计算该样本属于类别A和B的概率。

(2) 使用真实的联合概率，计算该样本属于类别A和B的概率。

(3) 讨论朴素贝叶斯的假设在这个例子中如何影响了分类结果。

**解答思路**：

设先验概率 $P(A) = P(B) = 0.5$。

**(1) 朴素贝叶斯（假设特征独立）**：

对于类别A：
$$\begin{aligned}
P(X_1=1, X_2=1|A) &= P(X_1=1|A) \times P(X_2=1|A) \\
&= 0.5 \times 0.5 \\
&= 0.25
\end{aligned}$$

对于类别B：
$$\begin{aligned}
P(X_1=1, X_2=1|B) &= P(X_1=1|B) \times P(X_2=1|B) \\
&= 0.5 \times 0.5 \\
&= 0.25
\end{aligned}$$

因此：
- $P(A|X_1=1, X_2=1) \propto 0.5 \times 0.25 = 0.125$
- $P(B|X_1=1, X_2=1) \propto 0.5 \times 0.25 = 0.125$

朴素贝叶斯预测：**两个类别概率相等，无法区分**。

**(2) 使用真实联合概率**：

对于类别A：
$$P(X_1=1, X_2=1|A) = 0.25$$

对于类别B：
$$P(X_1=1, X_2=1|B) = 0.5$$

因此：
- $P(A|X_1=1, X_2=1) \propto 0.5 \times 0.25 = 0.125$
- $P(B|X_1=1, X_2=1) \propto 0.5 \times 0.5 = 0.25$

归一化后：
$$P(B|X_1=1, X_2=1) = \frac{0.25}{0.125 + 0.25} = \frac{0.25}{0.375} = \frac{2}{3} \approx 0.667$$

正确分类：**类别B**。

**(3) 讨论**：

朴素贝叶斯假设特征独立，这在类别B的情况下是错误的。由于类别B中两个特征完全相关，$P(X_1=1, X_2=1|B)$ 的真实值是0.5，但朴素贝叶斯计算为0.25，低估了真实概率。这导致朴素贝叶斯无法区分A和B，而正确的方法应该倾向于类别B。

这个例子说明：
- 当特征之间存在强相关性时，朴素贝叶斯的独立性假设会导致概率估计偏差
- 但在许多实际应用中，即使特征相关，朴素贝叶斯仍然能给出正确的分类决策（只是概率值不准确）
- 在这个极端例子中，特征完全相关导致了分类失败

**(3) Python实现**

```python
from collections import defaultdict, Counter
import numpy as np

class SentimentNB:
    def __init__(self, alpha=1.0):
        self.alpha = alpha
        self.vocab = set()
        self.class_word_counts = defaultdict(Counter)
        self.class_total_counts = defaultdict(int)
        self.class_counts = defaultdict(int)
        
    def fit(self, texts, labels):
        for text, label in zip(texts, labels):
            words = text.split()
            self.class_counts[label] += 1
            for word in words:
                self.vocab.add(word)
                self.class_word_counts[label][word] += 1
                self.class_total_counts[label] += 1
        
    def predict(self, text):
        words = text.split()
        scores = {}
        
        for label in self.class_counts:
            # 对数先验
            score = np.log(self.class_counts[label] / sum(self.class_counts.values()))
            
            for word in words:
                # 拉普拉斯平滑
                word_count = self.class_word_counts[label].get(word, 0)
                smoothed_prob = (word_count + self.alpha) / \
                               (self.class_total_counts[label] + self.alpha * len(self.vocab))
                score += np.log(smoothed_prob)
            
            scores[label] = score
        
        return max(scores, key=scores.get)

# 训练数据
train_texts = [
    "这部电影太精彩了 演技出色",
    "非常好看 强烈推荐",
    "剧情紧凑 值得一看",
    "演员表演到位 制作精良",
    "这部电影太糟糕了 浪费时间",
    "非常无聊 不推荐",
    "剧情拖沓 看得想睡觉",
    "演员演技差 制作粗糙"
]
train_labels = ["正面", "正面", "正面", "正面", "负面", "负面", "负面", "负面"]

# 训练和预测
clf = SentimentNB(alpha=1.0)
clf.fit(train_texts, train_labels)

# 测试
test_texts = [
    "这部电影精彩 强烈推荐",
    "太无聊了 浪费时间"
]
expected = ["正面", "负面"]

for text, exp in zip(test_texts, expected):
    pred = clf.predict(text)
    status = "✓" if pred == exp else "✗"
    print(f"文本: {text}")
    print(f"预测: {pred}, 期望: {exp} {status}\n")
```
