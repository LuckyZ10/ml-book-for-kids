## 6.8 练习题

### 基础题

**练习 6.1**：距离计算

给定两个二维点 $A = (3, 4)$ 和 $B = (6, 8)$：

(a) 计算它们之间的欧氏距离

(b) 计算它们之间的曼哈顿距离

(c) 如果闵可夫斯基距离的 $p = 3$，结果是多少？

<details>
<summary>点击查看答案</summary>

(a) 欧氏距离：$d = \sqrt{(6-3)^2 + (8-4)^2} = \sqrt{9 + 16} = \sqrt{25} = 5$

(b) 曼哈顿距离：$d = |6-3| + |8-4| = 3 + 4 = 7$

(c) 闵可夫斯基距离：$d = (|6-3|^3 + |8-4|^3)^{1/3} = (27 + 64)^{1/3} = 91^{1/3} \approx 4.50$

</details>
