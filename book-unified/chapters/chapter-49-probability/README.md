## 49.1 引言：为什么机器学习需要概率论？

### 49.1.1 确定性的幻觉

想象你是一位气象学家。你收集了100年的降雨数据，发现每年3月15日有70%的概率会下雨。今天是3月15日，你会怎么预报？

**确定性思维**："根据历史，今天应该会下雨。"

**概率思维**："今天下雨的概率是70%，不下雨的概率是30%。我会带伞，但也不会取消户外活动。"

机器学习面对的是**不确定的世界**：
- 数据有噪声
- 模型有局限
- 未来不可预测

概率论给了我们描述和处理这种不确定性的数学语言。

## 49.2 概率基础：从硬币到贝叶斯

### 49.2.1 概率的三种解释

**频率派解释**：
- 概率是事件在重复试验中出现的频率
- "这枚硬币正面朝上的概率是50%" = 抛1000次，大约500次正面
- 适用于可重复的事件

**贝叶斯解释**：
- 概率是对信念程度的度量
- "明天下雨的概率是80%" = 我有80%的相信程度
- 适用于一次性事件

**公理化解释（科尔莫戈罗夫）**：
- 概率是满足特定公理的函数
- 不解释概率"是什么"，只规定它"如何运作"

### 49.2.2 基本概率公式

**加法法则**：
```
P(A ∪ B) = P(A) + P(B) - P(A ∩ B)
```
如果A和B互斥：P(A ∪ B) = P(A) + P(B)

**乘法法则**：
```
P(A ∩ B) = P(A) × P(B|A) = P(B) × P(A|B)
```
如果A和B独立：P(A ∩ B) = P(A) × P(B)

**贝叶斯定理**：
```
       P(B|A) × P(A)
P(A|B) = ─────────────
            P(B)
```

**费曼比喻**：
> 贝叶斯定理就像更新你的信念。一开始你可能认为某病很罕见（先验概率），但看到症状后（证据），你更新了对患病概率的估计（后验概率）。医生每天都在做贝叶斯推理，只是不用公式。

### 49.2.3 条件概率与独立性

**条件概率**：在已知某事件发生的情况下，另一事件发生的概率。

**例子**：
- 人群中患糖尿病的概率是10%
- 肥胖人群中患糖尿病的概率是30%
- P(糖尿病|肥胖) = 30%

**独立性**：两个事件独立意味着一个发生不影响另一个。

**例子**：
- 第一次抛硬币正面朝上，不影响第二次的结果
- P(第二次正面|第一次正面) = P(第二次正面) = 0.5

## 49.3 随机变量：用数字描述不确定性

### 49.3.1 什么是随机变量？

随机变量是一个**函数**，将随机试验的结果映射为数字。

**例子**：抛两枚硬币
- 结果空间：{正正, 正反, 反正, 反反}
- 随机变量 X = "正面朝上的次数"
- X(正正) = 2, X(正反) = 1, X(反正) = 1, X(反反) = 0

### 49.3.2 离散型随机变量

取值可数的随机变量。

**概率质量函数（PMF）**：
```
P(X = x) = 取值x的概率
```

**例子**：掷骰子
```
P(X=1) = 1/6, P(X=2) = 1/6, ..., P(X=6) = 1/6
```

### 49.3.3 连续型随机变量

取值不可数的随机变量（如实数区间）。

**概率密度函数（PDF）**：
```
f(x) ≥ 0
∫f(x)dx = 1
P(a ≤ X ≤ b) = ∫[a,b] f(x)dx
```

**注意**：对连续变量，P(X = x) = 0！只能谈区间概率。

**费曼比喻**：
> 概率密度就像云层的厚度。某一点的厚度不能告诉你有多少水，但一个区域的平均厚度乘以面积就能算出该区域的降水量。同样，概率密度在某点的值不是概率，只有积分后才成为概率。

### 49.3.4 累积分布函数（CDF）

```
F(x) = P(X ≤ x)
```

**性质**：
- F(-∞) = 0, F(+∞) = 1
- 单调不减
- 右连续

**离散与连续的关系**：
- 离散：CDF是阶梯函数
- 连续：CDF是光滑曲线，F'(x) = f(x)

## 49.4 常见分布：数据的概率模型

### 49.4.1 伯努利分布

**场景**：单次试验，两种结果（成功/失败）

```
P(X=1) = p      (成功)
P(X=0) = 1-p    (失败)
```

**应用**：
- 抛硬币
- 点击广告（点/不点）
- 邮件是否为垃圾邮件

**Python代码**：
```python
import numpy as np

# 伯努利分布：成功概率0.3
samples = np.random.binomial(n=1, p=0.3, size=1000)
print(f"成功次数: {samples.sum()}")
print(f"成功比例: {samples.mean():.3f}")
```

### 49.4.2 二项分布

**场景**：n次独立伯努利试验，成功次数

```
P(X=k) = C(n,k) × p^k × (1-p)^(n-k)
```

其中 C(n,k) = n!/(k!(n-k)!) 是组合数。

**性质**：
- 期望：E[X] = np
- 方差：Var(X) = np(1-p)

**应用**：
- 抛100次硬币，正面次数
- 1000个用户中，购买产品的数量

**Python代码**：
```python
import numpy as np
import matplotlib.pyplot as plt
from scipy import stats

# 二项分布：n=10, p=0.3
n, p = 10, 0.3
x = np.arange(0, n+1)
pmf = stats.binom.pmf(x, n, p)

plt.bar(x, pmf)
plt.xlabel('成功次数 k')
plt.ylabel('P(X=k)')
plt.title(f'二项分布 B(n={n}, p={p})')
plt.show()

# 计算概率
print(f"恰好3次成功的概率: {stats.binom.pmf(3, n, p):.4f}")
print(f"最多3次成功的概率: {stats.binom.cdf(3, n, p):.4f}")
```

### 49.4.3 正态分布（高斯分布）

**场景**：自然现象、测量误差、中心极限定理

**概率密度函数**：
```
            1           -(x-μ)²
f(x) = ───────── × exp(────────)
       σ√(2π)          2σ²
```

**记法**：X ~ N(μ, σ²)

**性质**：
- 关于μ对称
- σ控制"胖瘦"（标准差越大越分散）
- 68-95-99.7法则

**68-95-99.7法则**：
- P(μ-σ < X < μ+σ) ≈ 68%
- P(μ-2σ < X < μ+2σ) ≈ 95%
- P(μ-3σ < X < μ+3σ) ≈ 99.7%

**费曼比喻**：
> 正态分布就像抛一大把沙子的结果。大多数沙子落在中间（期望），少数散落在边缘（尾部）。无论原始分布是什么，只要叠加足够多的独立因素，结果就趋向正态分布——这就是大自然的平均力量。

**Python代码**：
```python
import numpy as np
import matplotlib.pyplot as plt
from scipy import stats

# 正态分布
mu, sigma = 0, 1
x = np.linspace(mu - 4*sigma, mu + 4*sigma, 100)
y = stats.norm.pdf(x, mu, sigma)

plt.plot(x, y, 'b-', linewidth=2)
plt.fill_between(x, y, alpha=0.3)
plt.xlabel('x')
plt.ylabel('概率密度 f(x)')
plt.title(f'正态分布 N(μ={mu}, σ={sigma})')
plt.axvline(mu, color='r', linestyle='--', label='μ')
plt.legend()
plt.show()

# 采样
samples = np.random.normal(mu, sigma, 10000)
print(f"样本均值: {samples.mean():.3f}")
print(f"样本标准差: {samples.std():.3f}")
```

### 49.4.4 均匀分布

**场景**：所有结果等可能

**离散均匀**：掷骰子
```
P(X=k) = 1/n,  k=1,2,...,n
```

**连续均匀**：U(a, b)
```
f(x) = 1/(b-a),  a ≤ x ≤ b
```

### 49.4.5 指数分布

**场景**：等待时间、寿命

**PDF**：
```
f(x) = λ × exp(-λx),  x ≥ 0
```

**性质**：
- 期望：E[X] = 1/λ
- 无记忆性：P(X>s+t|X>s) = P(X>t)

**应用**：
- 客服电话等待时间
- 放射性衰变时间
- 设备寿命

**Python代码**：
```python
import numpy as np
import matplotlib.pyplot as plt

# 指数分布：λ=0.5
lambda_param = 0.5
samples = np.random.exponential(1/lambda_param, 10000)

plt.hist(samples, bins=50, density=True, alpha=0.7)
plt.xlabel('等待时间')
plt.ylabel('概率密度')
plt.title(f'指数分布 (λ={lambda_param})')
plt.show()

print(f"理论期望: {1/lambda_param:.2f}")
print(f"样本均值: {samples.mean():.2f}")
```

### 49.4.6 泊松分布

**场景**：单位时间或空间内事件发生次数

```
         λ^k × exp(-λ)
P(X=k) = ─────────────
              k!
```

**性质**：
- 期望 = 方差 = λ
- 稀有事件的极限

**应用**：
- 每小时接到的电话数
- 每页书的错别字数
- 某路段每月交通事故数

## 49.5 期望与方差：预测的中心与波动

### 49.5.1 期望（均值）

**定义**：随机变量的"长期平均值"

**离散**：
```
E[X] = Σ x × P(X=x)
```

**连续**：
```
E[X] = ∫ x × f(x) dx
```

**性质**：
- E[aX + b] = aE[X] + b
- E[X + Y] = E[X] + E[Y]

### 49.5.2 方差与标准差

**方差**：偏离期望的"平均程度"
```
Var(X) = E[(X - E[X])²] = E[X²] - (E[X])²
```

**标准差**：σ = √Var(X)（与原数据同单位）

**性质**：
- Var(aX + b) = a²Var(X)
- 独立时：Var(X+Y) = Var(X) + Var(Y)

**费曼比喻**：
> 期望告诉你射手瞄准哪里，方差告诉你他的散布有多广。两个射手可能平均都打中靶心，但一个弹孔集中，一个四处飞散——方差告诉你谁更稳定。

### 49.5.3 协方差与相关性

**协方差**：两个变量一起变化的程度
```
Cov(X,Y) = E[(X-E[X])(Y-E[Y])]
```

**相关系数**：
```
           Cov(X,Y)
ρ(X,Y) = ─────────────
          σ_X × σ_Y
```

**性质**：
- -1 ≤ ρ ≤ 1
- ρ = 1：完全正相关
- ρ = -1：完全负相关
- ρ = 0：无线性相关（但可能有其他关系）

**Python代码**：
```python
import numpy as np

# 生成相关数据
np.random.seed(42)
n = 1000

# X和Y正相关
x = np.random.normal(0, 1, n)
y = 0.8 * x + np.random.normal(0, 0.6, n)

print(f"协方差: {np.cov(x, y)[0,1]:.3f}")
print(f"相关系数: {np.corrcoef(x, y)[0,1]:.3f}")

# 可视化
import matplotlib.pyplot as plt
plt.scatter(x, y, alpha=0.5)
plt.xlabel('X')
plt.ylabel('Y')
plt.title(f'正相关数据 (ρ={np.corrcoef(x,y)[0,1]:.2f})')
plt.show()
```

## 49.6 大数定律与中心极限定理

### 49.6.1 大数定律

**弱大数定律**：
> 随着样本量增加，样本均值依概率收敛于期望。

```
X̄_n → E[X]  (n→∞)
```

**直观理解**：
- 抛硬币，前10次可能有7次正面
- 抛1000次，大约500次正面
- 抛100万次，比例会非常接近50%

**意义**：让频率派概率成立的基础

### 49.6.2 中心极限定理（CLT）

**定理**：
> 独立同分布随机变量的和（或均值），在标准化后，依分布收敛于正态分布。

```
     X̄_n - μ
√n × ─────── → N(0,1)
       σ
```

**关键洞察**：
- 无论原始分布是什么（只要方差有限）
- 样本量足够大时，样本均值近似正态分布
- 解释了为什么正态分布无处不在

**Python演示**：
```python
import numpy as np
import matplotlib.pyplot as plt

# 演示中心极限定理
np.random.seed(42)

fig, axes = plt.subplots(2, 2, figsize=(12, 10))

distributions = [
    ('均匀分布 U(0,1)', lambda: np.random.uniform(0, 1, 1000)),
    ('指数分布 Exp(1)', lambda: np.random.exponential(1, 1000)),
    ('二项分布 B(10,0.3)', lambda: np.random.binomial(10, 0.3, 1000)),
    ('泊松分布 Poisson(2)', lambda: np.random.poisson(2, 1000))
]

for ax, (name, dist) in zip(axes.flat, distributions):
    # 每次取n个样本，计算均值，重复10000次
    sample_means = [np.mean(dist()) for _ in range(10000)]
    
    ax.hist(sample_means, bins=50, density=True, alpha=0.7, label='样本均值分布')
    ax.set_title(f'{name}\n样本量=1000, 重复10000次')
    ax.set_xlabel('样本均值')
    ax.set_ylabel('密度')
    ax.legend()

plt.tight_layout()
plt.show()
```

**费曼比喻**：
> 想象你把100个不同形状的积木扔进桶里摇晃。每个积木的形状可能很奇怪，但桶里积木的平均高度总是趋向一个钟形曲线。这就是中心极限定理的魔力——大自然喜欢平均，而平均的结果就是正态分布。

## 49.7 信息论基础：度量不确定性

### 49.7.1 信息量

**定义**：一个事件的信息量与其发生概率成反比
```
I(x) = -log₂P(x)
```

**单位**：比特（bit）

**例子**：
- "太阳从东边升起"：概率≈1，信息量≈0（废话）
- "明天下雨"：概率=0.5，信息量=1比特
- "中奖彩票号码"：概率=1/1000000，信息量≈20比特

### 49.7.2 信息熵

**定义**：随机变量的平均信息量，度量不确定性
```
H(X) = -Σ P(x) × log₂P(x)
```

**性质**：
- 均匀分布时熵最大
- 确定事件熵为0
- 熵越大，不确定性越大

**例子**：
- 公平硬币：H = 1比特
- 有偏硬币（90%正面）：H ≈ 0.47比特
- 确定性事件（100%正面）：H = 0

### 49.7.3 KL散度

**定义**：度量两个分布的差异
```
         P(x)
KL(P||Q) = Σ P(x) × log ────
         Q(x)
```

**性质**：
- KL ≥ 0，当且仅当P=Q时等于0
- 非对称：KL(P||Q) ≠ KL(Q||P)

**机器学习应用**：
- 变分自编码器（VAE）
- 策略优化（强化学习）
- 近似推断

**Python代码**：
```python
import numpy as np
from scipy import stats

# 计算熵
def entropy(p):
    p = np.array(p)
    p = p[p > 0]  # 避免log(0)
    return -np.sum(p * np.log2(p))

# 例子
fair_coin = [0.5, 0.5]
biased_coin = [0.9, 0.1]

print(f"公平硬币熵: {entropy(fair_coin):.3f} bits")
print(f"有偏硬币熵: {entropy(biased_coin):.3f} bits")

# KL散度
p = [0.5, 0.5]
q = [0.9, 0.1]
kl_div = np.sum(np.array(p) * np.log2(np.array(p) / np.array(q)))
print(f"KL(P||Q): {kl_div:.3f} bits")
```

## 49.8 概率在机器学习中的应用

### 49.8.1 最大似然估计（MLE）

**思想**：选择使观测数据最可能的参数

**例子**：抛硬币10次，7次正面
- 假设正面概率p
- 似然函数：L(p) = p⁷(1-p)³
- 最大化L(p) → p̂ = 0.7

**与损失函数的关系**：
- 负对数似然 = 交叉熵损失
- 最小化损失 = 最大化似然

### 49.8.2 最大后验估计（MAP）

**思想**：结合先验知识和观测数据
```
P(θ|D) ∝ P(D|θ) × P(θ)
```

**与正则化的关系**：
- 高斯先验 ↔ L2正则化
- 拉普拉斯先验 ↔ L1正则化

### 49.8.3 贝叶斯机器学习

**不是点估计，而是分布**：
- 先验 P(θ)
- 似然 P(D|θ)
- 后验 P(θ|D)

**优势**：
- 天然的不确定性量化
- 避免过拟合
- 小样本场景更有效

### 49.8.4 概率图模型

**用图表示变量间的概率关系**：
- 贝叶斯网络（有向图）
- 马尔可夫随机场（无向图）

**应用**：
- 隐马尔可夫模型（HMM）
- 条件随机场（CRF）
- 主题模型（LDA）

## 49.9 本章小结

### 核心概念回顾

**1. 概率基础**
- 概率是频率或信念的度量
- 条件概率：P(A|B) = P(A∩B)/P(B)
- 贝叶斯定理：更新信念的数学公式

**2. 随机变量**
- 离散型：概率质量函数（PMF）
- 连续型：概率密度函数（PDF）
- 累积分布函数（CDF）：P(X≤x)

**3. 重要分布**
| 分布 | 场景 | 关键参数 |
|------|------|---------|
| 伯努利 | 二元事件 | p |
| 二项 | n次伯努利 | n, p |
| 正态 | 自然现象、误差 | μ, σ |
| 指数 | 等待时间 | λ |
| 泊松 | 稀有事件计数 | λ |

**4. 数字特征**
- 期望：E[X]，长期平均值
- 方差：Var(X)，离散程度
- 协方差/相关系数：变量间关系

**5. 极限定理**
- 大数定律：样本均值→期望
- 中心极限定理：均值分布→正态

**6. 信息论**
- 熵：不确定性度量
- KL散度：分布差异度量

### 费曼法一句话总结

> 概率论是给不确定性定量的语言。它告诉我们世界不是确定的，但不确定本身可以被精确描述。从抛硬币到天气预报，从医学诊断到股票预测，概率思维让我们在不完美的信息中做出最好的决策。

## 49.10 练习题

### 基础练习

**练习1：条件概率**

某疾病在人群中的患病率是1%。检测的准确率为：
- 患者检测阳性率：99%
- 健康者检测阴性率：99%

如果一个人检测为阳性，他实际患病的概率是多少？

<details>
<summary>点击查看答案</summary>

使用贝叶斯定理：

P(病|阳) = P(阳|病) × P(病) / P(阳)

P(阳) = P(阳|病)P(病) + P(阳|健康)P(健康)
      = 0.99×0.01 + 0.01×0.99
      = 0.0198

P(病|阳) = 0.99×0.01 / 0.0198 ≈ 0.5 = 50%

**反直觉！** 即使检测准确率99%，由于疾病罕见，阳性结果只有50%的准确率。这就是假阳性的影响。

</details>

**练习2：期望与方差**

掷一个公平的六面骰子：
(1) 计算期望
(2) 计算方差
(3) 掷两次，和的期望和方差是多少？

<details>
<summary>点击查看答案</summary>

(1) 期望：
E[X] = (1+2+3+4+5+6)/6 = 3.5

(2) 方差：
E[X²] = (1+4+9+16+25+36)/6 = 91/6 ≈ 15.17
Var(X) = E[X²] - (E[X])² = 15.17 - 12.25 = 2.92

(3) 两次掷骰子：
E[X+Y] = E[X] + E[Y] = 7
Var(X+Y) = Var(X) + Var(Y) = 5.84 (独立)

</details>

**练习3：正态分布**

某班级考试成绩服从N(75, 100)：
(1) 分数在65-85之间的概率是多少？
(2) 多少分能进入前5%？

<details>
<summary>点击查看答案</summary>

μ=75, σ=10

(1) 65 = μ-σ, 85 = μ+σ
根据68-95-99.7法则，概率≈68%
精确值：P(65<X<85) ≈ 0.6827

(2) 查标准正态表，前5%对应z≈1.645
分数 = μ + z×σ = 75 + 1.645×10 ≈ 91.45

需要约92分才能进入前5%。

</details>
